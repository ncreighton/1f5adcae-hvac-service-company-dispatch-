== Description ==

A WordPress plugin for HVAC contractors, plumbing companies, and field service businesses managing technician dispatch and maintenance contracts. Features a service territory map with technician location tracking, work order assignment with skill-matching routing, preventive maintenance contract billing with automated renewal invoicing, parts inventory management with vendor reorder alerts, customer portal for service history and equipment records, and a technician mobile interface for job completion signatures and photo uploads.

**Features:**

* Core functionality

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via WordPress admin > Plugins
3. Configure via Settings > HVAC Service Company Dispatch and Preventive Maintenance WordPress Plugin
